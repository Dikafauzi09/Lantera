LENTERA adalah platform website berbasis peta (map-based civic platform) yang berfungsi sebagai media pelaporan, pemantauan, dan informasi kondisi lingkungan serta fasilitas publik, terinspirasi dari aplikasi JAKI (Jakarta Kini).
Website ini mengintegrasikan data IoT dan laporan warga untuk membantu pemerintah atau pengelola wilayah dalam mengambil keputusan berbasis data.
